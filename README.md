# 💖 爱的守护 💖

一个浪漫的HTML5小游戏，通过控制粉色小鱼收集爱心，躲避障碍，体验守护与成长的故事。

## 🎮 游戏简介

在这个充满爱意的海洋中，你扮演一条粉色小鱼，在聪仔（蓝色守护鱼）的陪伴下，收集代表爱意的❤️爱心，躲避各种困难和挑战。随着分数的增加，你会遇到更大的挑战，但也会获得更珍贵的奖励。

游戏以"守护"为主题，通过精美的视觉效果、丰富的游戏机制和温馨的文案，讲述了一段关于爱与守护的浪漫故事。

## 🎯 游戏玩法

### 基本操作
- **移动控制**：鼠标移动或触摸屏幕控制小鱼移动
- **收集爱心**：吃掉粉色的❤️爱心增加恋爱指数和心动值
- **躲避障碍**：避开各种颜色的障碍鱼，它们会阻碍你的成长
- **使用道具**：收集特殊道具获得增益效果
- **暂停游戏**：点击右上角暂停按钮随时暂停

### 游戏目标
- 尽可能收集更多的爱心，提高恋爱指数
- 在困难模式下，大鱼会从四面八方出现并主动追击
- 收集心动值，触发无敌爆发模式
- 挑战更高的分数，解锁更多障碍类型

## 🌟 游戏特色

### 三种难度模式

#### 🍰 甜蜜约会（简单模式）
- **适合人群**：新手玩家
- **特点**：
  - 爱心出现频率高
  - 障碍鱼较少且速度较慢
  - 道具出现概率较高
  - 大鱼不会主动追击
  - 心动值持续时间：5秒
  - 守护持续时间：15秒
  - 聪仔出击持续时间：8秒

#### 🏠 幸福日常（普通模式）
- **适合人群**：有一定游戏经验的玩家
- **特点**：
  - 平衡的游戏体验
  - 中等数量的障碍和道具
  - 大鱼在困难模式下会主动追击
  - 心动值持续时间：4秒
  - 守护持续时间：12秒
  - 聪仔出击持续时间：6秒

#### 🛡️ 守护一生（困难模式）
- **适合人群**：追求挑战的玩家
- **特点**：
  - 障碍鱼从四面八方出现
  - 大鱼会主动追击玩家（靠近时）
  - 道具出现概率较低
  - 迷雾能见度极低
  - 大鱼出现概率随分数动态调整
  - 心动值持续时间：3秒
  - 守护持续时间：8秒
  - 聪仔出击持续时间：4秒

### 丰富的鱼类品种

游戏中有多种精美的鱼类，每种都有独特的造型、颜色和动画效果：

#### 友好鱼类
- **粉色小鱼（玩家）**：你扮演的角色，随着成长会逐渐变大
- **蓝色守护鱼（聪仔）**：你的守护者，可以为你抵挡伤害

#### 障碍鱼类
- **金鱼**：橙色，大而飘逸的尾巴，从早期到后期都有不同大小
- **热带鱼**：粉红色，分叉尾巴，带彩色斑点
- **小丑鱼**：橙红色，带黑色条纹，可爱但危险
- **海豚**：蓝色，流线型身体，带水花效果
- **天使鱼**：金色，高而扁的身体，优雅但致命
- **鲸鱼**：深灰色，巨大的体型，移动缓慢但威胁巨大
- **鲨鱼**：红色，快速且危险，后期出现的终极挑战

### 特殊事件系统

游戏会随机触发特殊事件，增加游戏趣味性和挑战性：

#### 🌫️ 迷茫之雾
- **效果**：能见度大幅降低，只能看到周围很小的范围
- **提示**：靠近聪仔时能见度会提高
- **策略**：在迷雾中要格外小心，慢慢移动

#### 🌊 命运激流
- **效果**：强大的水流推动玩家和所有物体
- **提示**：需要逆流而上，考验操作技巧
- **策略**：顺着水流方向移动，利用水流躲避障碍

#### 🌠 幸运流星
- **效果**：大量幸运星从天而降
- **提示**：收集星星可以获得额外分数
- **策略**：抓住机会收集更多星星

### 道具系统

#### 🍬 甜蜜糖果
- **效果**：提升移动速度
- **持续时间**：5秒（300帧）
- **图标**：🍬

#### 🛡️ 护身符（聪仔守护）
- **效果**：召唤聪仔守护在身边，可以抵挡一次伤害
- **持续时间**：根据难度不同（简单15秒/普通12秒/困难8秒）
- **特殊机制**：
  - 守护时间结束后，聪仔会游走并显示"聪仔去打工了~"
  - 如果玩家受到伤害，聪仔会消失
- **图标**：🛡️

#### 🧲 双向奔赴
- **效果**：自动吸引附近的爱心和道具
- **持续时间**：约6.7秒（400帧）
- **吸引范围**：200像素
- **图标**：🧲

#### 🎯 聪仔出击
- **效果**：让聪仔主动出击，自动追击并吃掉小型障碍鱼
- **持续时间**：根据难度不同（简单8秒/普通6秒/困难4秒）
- **特殊机制**：
  - 如果聪仔已经激活守护，使用后会主动吃小鱼，时间到了回到守护状态
  - 如果聪仔未激活，使用后会激活并吃小鱼，时间到了会消失（除非中途吃到守护道具）
  - 如果玩家受到伤害，聪仔会消失
- **图标**：🎯
- **出现条件**：分数达到200分以上

#### 💎 钻石（特殊爱心）
- **效果**：直接填满心动值，触发无敌爆发模式
- **稀有度**：非常稀有
- **图标**：💎

### 心动值系统

收集爱心可以积攒心动值，当心动值满100%时，会触发**🔥 恋爱爆发**模式：

- **无敌状态**：短时间内完全无敌，可以吃掉所有障碍
- **自动吸引**：自动吸引所有爱心和道具
- **持续时间**：根据难度不同（简单5秒/普通4秒/困难3秒）
- **视觉效果**：屏幕背景变为粉红色，玩家周围有光晕效果

### 连击系统

- **机制**：在2秒内连续收集道具会累计连击数
- **显示**：屏幕顶部会显示连击数（如"3 连击!"）
- **效果**：连击数越高，视觉效果越明显
- **提示**：保持连击可以获得更好的游戏体验

## 🎨 视觉效果

### 基础视觉
- **精美的鱼类绘制**：每种鱼都有独特的造型、颜色渐变和动画效果
- **渐变色彩系统**：随着分数变化，背景颜色会动态改变
  - 0-500分：粉色渐变（甜蜜期）
  - 500-1000分：紫色渐变（浪漫期）
  - 1000-2000分：深蓝渐变（挑战期）
  - 2000分以上：黑色渐变（终极挑战）
- **流畅的动画**：所有游戏对象都有平滑的移动和旋转动画

### 特效系统

#### 粒子特效
- **收集特效**：收集道具时会产生粒子爆炸效果
- **粒子类型**：星星形和圆形粒子
- **颜色变化**：粒子颜色随道具颜色变化
- **动画效果**：粒子会旋转、扩散并逐渐消失

#### 屏幕震动
- **触发条件**：玩家受到伤害时
- **效果**：屏幕会震动，增强打击感
- **强度**：根据伤害类型调整震动强度

#### 分数动画
- **效果**：收集道具时显示"+分数"动画
- **动画**：分数文字向上飘动并逐渐消失
- **颜色**：与道具颜色一致

#### 背景美化
- **光斑效果**：8个光斑在背景中缓慢移动并闪烁
- **气泡系统**：25个气泡从底部上升，带有高光和水平漂移效果
- **动态背景**：根据游戏阶段和事件改变背景效果

### 文字提示
- **情话系统**：收集爱心时会随机显示温馨情话
- **事件提示**：特殊事件触发时会显示提示文字
- **状态提示**：获得道具或触发效果时会显示状态文字

## 💡 游戏提示与策略

### 开局策略
1. **从小开始**：开局时先从小鱼开始，慢慢长大后再挑战大鱼
2. **优先收集**：优先收集粉色爱心，快速积攒心动值
3. **保持距离**：与障碍鱼保持安全距离，特别是大鱼

### 道具使用策略
1. **护身符**：困难模式下护身符出现概率很低，但非常珍贵，要善加利用
2. **聪仔出击**：在障碍鱼较多时使用，可以快速清理小型障碍
3. **双向奔赴**：在爱心较多时使用，可以快速收集
4. **甜蜜糖果**：在需要快速移动躲避时使用

### 特殊事件应对
1. **迷雾事件**：迷雾降临时，尽量靠近聪仔，能见度会更高
2. **激流事件**：顺着水流方向移动，利用水流躲避障碍
3. **流星事件**：抓住机会收集更多星星

### 困难模式技巧
1. **大鱼追击**：困难模式下，大鱼会在你靠近时主动追击，保持距离很重要
2. **四面受敌**：障碍鱼会从四面八方出现，要时刻注意周围
3. **动态难度**：大鱼出现概率随分数动态调整，早期不要过于激进
4. **道具珍贵**：困难模式下道具更珍贵，要合理使用

### 进阶技巧
1. **连击保持**：保持连击可以获得更好的视觉效果和游戏体验
2. **心动值管理**：合理积攒心动值，在关键时刻触发无敌模式
3. **聪仔配合**：善用聪仔的守护和出击功能，可以大大提高生存率
4. **阶段规划**：了解不同分数阶段的特点，提前做好准备

## 🎯 游戏机制详解

### 成长系统
- **大小变化**：随着分数增加，玩家会逐渐变大（最大到60像素）
- **能力提升**：变大后可以吃掉更小的障碍鱼
- **视觉反馈**：大小变化有平滑的动画过渡

### 动态难度
- **分数阈值**：达到特定分数会解锁新的障碍类型
  - 500分：争吵与冷战
  - 1000分：嫉妒
  - 2000分：巨大的现实压力
  - 3000分：情感危机
- **难度递增**：随着分数提高，会出现更大的挑战
- **大鱼概率**：困难模式下，大鱼出现概率随分数动态调整

### 碰撞检测
- **圆形碰撞**：使用圆形碰撞检测，精确判断碰撞
- **大小判断**：玩家只能吃掉比自己小的障碍鱼
- **守护机制**：聪仔可以抵挡一次伤害

### 全屏适配
- **响应式设计**：完美支持移动端和桌面端
- **触摸优化**：针对触摸操作进行了优化
- **安全区域**：支持iPhone等设备的安全区域适配
