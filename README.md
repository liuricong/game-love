# 💖 爱的守护 💖

一个浪漫的HTML5小游戏，通过控制粉色小鱼收集爱心，躲避障碍，体验守护与成长的故事。

## 🎮 游戏简介

在这个充满爱意的海洋中，你扮演一条粉色小鱼，在聪仔（蓝色守护鱼）的陪伴下，收集代表爱意的❤️爱心，躲避各种困难和挑战。随着分数的增加，你会遇到更大的挑战，但也会获得更珍贵的奖励。

## 🎯 游戏玩法

### 基本操作
- **移动控制**：鼠标移动或触摸屏幕控制小鱼移动
- **收集爱心**：吃掉粉色的❤️爱心增加恋爱指数
- **躲避障碍**：避开各种颜色的障碍鱼，它们会阻碍你的成长
- **使用道具**：收集特殊道具获得增益效果

### 游戏目标
- 尽可能收集更多的爱心，提高恋爱指数
- 在困难模式下，大鱼会从四面八方出现并主动追击
- 收集心动值，触发无敌爆发模式

## 🌟 游戏特色

### 三种难度模式
- **🍰 甜蜜约会（简单）**：适合新手，爱心多，障碍少
- **🏠 幸福日常（普通）**：平衡模式，体验完整的游戏内容
- **🛡️ 守护一生（困难）**：挑战模式，大鱼从四面八方出现并主动追击

### 丰富的鱼类品种
游戏中有多种精美的鱼类：
- **金鱼**：橙色，大而飘逸的尾巴
- **热带鱼**：粉红色，分叉尾巴，带斑点
- **小丑鱼**：橙红色，带黑色条纹
- **海豚**：蓝色，流线型身体，带水花效果
- **天使鱼**：金色，高而扁的身体
- **鲸鱼**：深灰色，巨大的体型
- **鲨鱼**：红色，快速且危险

### 特殊事件
- **🌫️ 迷茫之雾**：能见度大幅降低，需要更小心
- **🌊 命运激流**：强大的水流推动，考验操作
- **🌠 幸运流星**：大量幸运星从天而降

### 道具系统
- **🍬 甜蜜糖果**：提升移动速度
- **🛡️ 护身符**：召唤聪仔守护，可以抵挡一次伤害
- **🧲 双向奔赴**：自动吸引附近的爱心和道具

### 心动值系统
收集爱心可以积攒心动值，当心动值满100%时，会触发**🔥 恋爱爆发**模式：
- 短时间内无敌状态
- 可以吃掉所有障碍
- 自动吸引所有爱心和道具

## 🎨 视觉效果

- 精美的鱼类绘制，每种鱼都有独特的造型和动画
- 渐变色彩系统，随着分数变化背景颜色
- 光晕和水花特效
- 流畅的动画和粒子效果

## 💡 游戏提示

1. **开局策略**：先从小鱼开始，慢慢长大后再挑战大鱼
2. **护身符**：困难模式下护身符出现概率很低，但非常珍贵
3. **迷雾事件**：迷雾降临时，尽量靠近聪仔，能见度会更高
4. **大鱼追击**：困难模式下，大鱼会在你靠近时主动追击，保持距离很重要
5. **心动值**：优先收集粉色爱心，快速积攒心动值触发无敌模式

## 🎯 游戏机制

- **成长系统**：随着分数增加，玩家会逐渐变大
- **动态难度**：随着分数提高，会出现更大的挑战
- **阶段解锁**：达到特定分数会解锁新的障碍类型
- **全屏适配**：完美支持移动端和桌面端
